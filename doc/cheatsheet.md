# dongbei 语言考试小抄

## 变量

```
老张是活雷锋。  # 定义变量。初始值：啥也不是。
老张装二。  # 老张 = 2
削老张。  # 老张 = 啥也不是
老王是活雷锋。
老王装仨。  # 老王 = 3
老张装老王。  # 引用变量。老张 = 3
```

## 运算

```
老张走走。  # 老张 += 1
老张走两步。  # 老张 += 2
老张稍稍。  # 老张 -= 1
老张稍三步。  # 老张 -= 3
老张装老王加二。  # 老张 = 老王 + 2
老张装（老张减三）除以五乘老王。  # 老张 = (老张 - 3)/5*老王
```

## 比较

```
老王比5大

```

## 打印

```
唠唠：“唉呀妈呀”。  # 打印字符串常量。
唠唠：老王。  # 打印变量的值。
唠唠：老王比老张大。  # 打印表达式的值。
唠唠：“老王是”、老王。  # 打印两个表达式的并置。
```

## 条件

```
寻思：老王比老张大？
要行咧就唠唠：“老王！”。
要不行咧就唠唠：“老张！”。
```

## 循环

```
老王从1到10磨叽：
  唠唠：老王。
磨叽完了！
```

## 组合

```
寻思：老王比老张大？
要行咧就开整：
  唠唠：“老王！”。
  唠唠：“你好美！”。
整完了。
```
## 套路

```
【阶乘】（那啥）咋整：  # 定义套路 阶乘，有一个参数 那啥。
  寻思：那啥比一小？ # 需要自推吗？
  要行咧就 滚犊子吧 一。  # 返回值1。
  要不行咧就 滚犊子吧 那啥乘整【阶乘】（那啥减一）。  # 需要。返回自推结果。
整完了。  # 定义结束。

唠唠：整【阶乘】（五）！  # 使用套路
```
